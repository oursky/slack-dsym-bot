# Slack-dsym-bot

Slack bot for iOS crash report via Sentry 
* for more, please read (this post)[http://code.oursky.com/slack-dsym-bot-report-via-sentry]

## SETUP
* Fill in config settings in `config.js`
* Start with `node index.js`
